# traffic-analytics-project
Smart City Traffic &amp; Accident Analytics System: An end-to-end Data Analytics project that analyzes traffic patterns, detects accident hotspots, predicts accident risk, and provides interactive insights via a Streamlit dashboard using Python, ML, and visualization tools

## Project Objective
Analyze city traffic data, identify congestion patterns, detect accident hotspots, and predict accident risk.

## Project Structure
- `data/`: Contains raw and processed datasets.
- `src/`: Python scripts for data generation, cleaning, EDA, hotspot detection, and ML training.
- `app/`: Streamlit dashboard.
- `requirements.txt`: Python dependencies.

## Setup
1. Install dependencies:
   `pip install -r requirements.txt`
2. Run data pipeline:
   `python run_pipeline.py`
3. Start Dashboard:
   `streamlit run app/traffic_dashboard.py`
